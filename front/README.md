# Portfólio - Matheus Hamada

Landing page pessoal desenvolvida com Next.js, React, TypeScript e Tailwind CSS.

## Como executar

Instale as dependências:

```bash
npm ci
```

Inicie o servidor de desenvolvimento:

```bash
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000) no navegador.

## Scripts disponíveis

```bash
npm run lint
npm run build
npm run start
```

## Estrutura

- `app/page.tsx`: composição principal da landing page.
- `app/_components`: componentes reutilizáveis e seções da página.
- `app/_data/site.ts`: textos, links, habilidades, experiências e certificações.
- `public/.gitkeep`: mantém o diretório `public` versionado para o deploy.
